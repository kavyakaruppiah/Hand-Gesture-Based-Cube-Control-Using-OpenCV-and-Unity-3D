🖐️ Hand Gesture Controlled Cube Movement

(OpenCV + Unity 3D Integration)

📌 Project Overview

This project presents a real-time hand gesture control system where a cube in Unity 3D moves based on the user’s hand movements. Using a webcam, hand gestures are detected and translated into directional controls without any physical input device.

🎯 Objective

To develop a touchless interaction system that allows users to control virtual objects using natural hand gestures.

⚙️ Technologies Used
Unity 3D
OpenCV
MediaPipe / Hand Tracking
Python
🧠 System Working
The webcam captures live hand movements.
Hand gestures are detected and tracked in real time.
The system identifies the direction of the hand.
Corresponding movement commands are generated.
The cube in Unity moves based on these commands.
✋ Gesture-Based Controls
Hand moves left → Cube moves left
Hand moves right → Cube moves right
Hand moves up → Cube moves forward
Hand moves down → Cube moves backward
