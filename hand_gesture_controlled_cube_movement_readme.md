# 🖐️ Hand Gesture Controlled Cube Movement (OpenCV + Unity 3D)

## 📌 Project Overview
This project implements a real-time interaction system where a cube in Unity 3D is controlled using hand gestures. The system captures hand movements through a webcam, processes them using computer vision, and translates them into directional movements of a cube in a virtual environment.

## 🎯 Objective
To design and develop a touchless control system that enables users to interact with virtual objects using natural hand gestures without the need for physical controllers.

## ⚙️ Technologies Used
- Unity 3D
- OpenCV
- MediaPipe (Hand Tracking)
- Python

## 🧠 System Architecture
The system consists of two main components:
1. **Hand Gesture Detection Module** – Detects and tracks hand movements using a webcam.
2. **Unity Visualization Module** – Receives gesture data and controls cube movement.

## 🔄 Working Methodology
1. The webcam captures live video input.
2. Hand gestures are detected and tracked in real time.
3. The position and movement of the hand are analyzed.
4. Directional commands are generated based on hand movement.
5. These commands are transmitted to the Unity environment.
6. The cube moves accordingly in real time.

## ✋ Gesture Controls
- Move hand left → Cube moves left
- Move hand right → Cube moves right
- Move hand up → Cube moves forward
- Move hand down → Cube moves backward
